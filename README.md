# Nyancat YouTube Play Bar
A cute little Chrome Extension to turn your YouTube play bar into Nyancat!

As Google has decided I'm not old enough to put things on the Chrome Web Store, here are some
# Chrome Install Instructions
## 1: Download or clone the repository
```bash
git clone https://github.com/redd-rl/nyan-cat-youtube-play-bar.git
```
Or click `<> Code` up above and click `Download ZIP`
## 2: Extract the extension
If you downloaded a ZIP, extract it somewhere you think will be good.
If you git cloned, skip to step 3.
## 3: Open Chrome's Extensions page.
Navigate to `chrome://extensions/` in Chrome.
## 4: Enable Developer Mode
Click the little switch in the top right.
## 5: Load the extension
Click **Load unpacked** and select the folder where you extracted or cloned the extension to.
## 6: Done!
It should now work!

# Common issues
## Failed to load extension - Manifest file is missing or unreadable. Could not load manifest
Look inside the nyan-cat-youtube-play-bar-main directory, is there another folder inside?
If so, load that folder instead.
If not, make an issue on this GitHub repository.
